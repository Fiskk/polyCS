Connect to MultiPoint Server using OSx


Starting with OSX Snow Leopard, the Microsoft Remote Desktop app is available in the Mac App Store. This app can be found in modified form for iOS devices as well. Most of the same steps apply there as well.


Using the App


1. Open the Microsoft Remote Desktop app and click New



2. Enter the credentials for the server you are connecting to. Fill in the fields and credentials as necessary. You may also want to change the Colors to something higher if you want a nice-looking display.



3. To redirect folders to the remote machine, change to the Redirection tab and click the plus sign in the lower left corner. Add the folders you want to connect and return to the main menu.



4. Now double click on the connection and the connection to the server will start.
