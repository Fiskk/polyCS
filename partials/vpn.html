A Virtual Private Network (VPN) extends a private network (in this case, the CS-Department's network) over the Internet. Using a VPN allows a user to securely access the necessary information within a private network over the Internet as if they never left the initial network.

1. Before beginning, you must have an activated CS account. If you don't have one, you can sign up for one at http://accounts.cs.sunyit.edu/activate.php.


2. Next, you'll need a VPN client. For this tutorial, we assume that you are using OpenVPN, but the process will be similar with other clients as well.

Windows users and OSx users can download OpenVPN's installer at https://openvpn.net/ Users of Debian-based Linux distros will have the command-line client installed by default. If your distro lacks OpenVPN, you can grab a tarball of the source at https://openvpn.net/index.php/download/community-downloads.html



To install OpenVPN, Windows or OSx users can run their respective installer-files and click through their respective wizards. Linux users will have to configure their client via command-line. A good tutorial is available over at Linux.com



3. You're also going to need to download some files that are available here. Extract the files. You'll need them later.



4. Launch OpenVPN. At this point Linux users will have to refer once again to Linux.com since the remainder of this tutorial only demonstrates use of the the GUI version of this software.



5. Once OpenVPN is active, click the plus sign next to Connection Profiles.



6. Select Local File and click Import.
